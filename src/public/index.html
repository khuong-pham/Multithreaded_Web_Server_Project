Project Overview

A high-performance HTTP server built from scratch using modern C++17 features. This project demonstrates systems programming concepts including network programming, concurrent processing, and protocol implementation.

The server handles HTTP/1.1 requests, serves static files with proper MIME type detection, and implements robust error handling. Currently supporting single-threaded operations with multithreading implementation in progress.

Technical Architecture

Core Architecture

  • Modular Design: Separation of concerns with dedicated modules
  • RAII Pattern: Automatic resource management
  • Exception Safety: Comprehensive error handling
  • Security: Path traversal protection

Core Components

  • Server Class: Connection Handler and Worker Manager
  • HTTP Parser: Request parsing and validation
  • File Handler: Static file serving with MIME detection
  • Request Router: URL-based request routing

Performance Features

  • Memory Efficient: Minimal memory usage
  • Fast Response: Optimized static file serving
  • LRU Caching: In-memory cache with LRU eviction
  • Thread Pool: Scalable multithreaded processing
